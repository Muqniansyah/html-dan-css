disini adalah contoh dari hyperlink jenis page anchor

kesimpulannya : hyperlink memiliki atribut yang bernama href dan href memiliki 3 jenis yaitu external link, internal link/ relative url, dan page anchor.

external link = link ke website lain

internal link = ke1 domain, ketika ingin masuk ke file yang lain kita dengan link

relative url = 1) ketika satu file kita ada difolder tambahan yang dibuat dan 1 file lagi tidak memakai folder tapi disuatu folder yang sama.

2) ketika file satu dan dua ada di folder tambahan baru yang berbeda beda tapi masih dalam satu folder yang sama.

page anchor = ini untuk bisa mencet link dan langsung terhubung kebagian tertentu dalam 1 halaman.
