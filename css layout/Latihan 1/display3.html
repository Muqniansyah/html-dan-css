definisi dari display block

  1. elemen html yg secara default menammbahkan baris baru.
  2. jika tidak diatur lebarnya, maka lebar default dari elemen block akan memenuhi lebar dari browser/parentnya.
  3. kita dapat mengatur tinggi dan lebar dari elemen block.
  4. didalam elemen block, kita dapat menyimpan tag dengan elemen inline, inline-block, atau bahkan elemen block lagi.
