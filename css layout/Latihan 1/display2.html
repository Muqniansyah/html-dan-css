definisi dari display inline-block

1. tidak ada elemen yang secara default memiliki properti dispay=inline-block

2. kita harus ubah secara manual properti tersebut.

3. perilaku dasarnya sama dengan elemen inline.

4. perbedaannya, elemen inline-block dapat kita atur tinggi dan lebarnya.

5. biasanya digunakan untuk navigasi.
