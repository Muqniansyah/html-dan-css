ini adalah penjelasan tentang clear.

cara menghentikan/membersihkan float :

  1. menggunakan properti overflow
  2. menggunakan div kosong
  3. menggunakan teknik micro clearfix

clear berfungsi menghentikan/membersihkan float.pastikan jika menggunakan float, kalau tidak butuh lagi, harus hentikan /bersihkan floatnya.dan valuenya dari clear ada 3, yaitu :

  1. left : untuk yang kiri
  2. right : untuk yang kanan
  3. both : untuk keduanya/kanan dan kiri

jadi clear berfungsi untuk menghentikan /bersihkan float yang sudah dibuat. cara bersihkannya dengan menggunakan div kosong diakhir floatnya.
