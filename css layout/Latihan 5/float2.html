ini adalah contoh dari out-of flow

float merupakan properti pada css untuk mengatur posisi sebuah elemen. sebuah elemen dapat dipaksa untuk berada disebelah kiri/kanan dari parent/pembukusnya dengan menggunakan properti ini.
value dari float ada 3 jenis, yaitu : 1.) none :defaultnya. 2.) left :elemennya akan berada dikiri. 3.) right:elemennya akan berada dikanan.
float termasuk tipe out-of flow


penjelasannya :kotak biru hilang, ada dibelakang kotak hijau. karna jika dikasih float, kotak yang biru menganggap kotak hijau gak ada,karna kotak hijau sudah keluar dari flow normal.

kotak yang diberikan float,akan keluar dari flow normalnya.

jika diberikan float right,maka akan dipaksa disebelah kanan elemen pembungkusnya.


perilaku dari elemen elemen ketika diberikan css:

  1. normal flow : yang normal
  2. out-of flow
