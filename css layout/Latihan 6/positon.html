css position sangat penting, karna akan membuat kalian dapat melakukan tata letak dari elemen elemen html yang kalian buat sesuai dengan yang diinginkan.

position memungkinkan kita untuk menggeserkan sebuah elemen tanpa memengaruhi elemen disekitarnya.

untuk menggunakan css position kita perlu sebuah properti yang namanya position, dan valuenya ada 4. yaitu :

  1. static : defaultnya
  2. relative : non static
  3. absolute : non static
  4. fixed : non static

konsep css position

  1. static adalah nilai default dari tiap tiap elemen ketika tidak diberi properti position.
  2. menggunakan position selain static(non static), akan membuat elemen menjadi seolah olah berbeda dimensi dari elemen lainnya.
  3. elemen yang diberi position selain static dapat menggunakan properti top, left, bottom, dan right untuk mengatur posisinya.
  4. jadi position intinya, yang tadinya ada dalam flow normal dibuat keluar dari flow/out of flow. anggap saja maju 1 dimensi.
