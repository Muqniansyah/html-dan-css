konsep

konsepnya yaitu :

  1. Setiap elemen dihalaman berada didalam sebuah box(kotak).
  2. Kita bisa mengatur ukuran dan posisi kotak tersebut.
  3. kita bisa memberi warna/gambar sebagai background kotak tersebut.


css box model mendefinisikan 'kotak' yang dihasilkan oleh sebuah elemen, lalu menampilkannya sesuai dengan format visualnya.

css box model terdiri dari :

  1. margin

    2. area transparan disekitar kotak(diluar border)

  2. padding

    3. area transparan didalam kotak (antara content dan border)

  3. border

    4. batas disekeliling content dan padding

  4. content

    5. konten sebenarnya didalam box, bisa berupa teks/gambar.



properti untuk margin :

  1. margin-top
  2. margin-right
  3. margin-bottom
  4. margin-left
  5. margin

properti untuk padding :

  1. padding-top
  2. padding-right
  3. padding-bottom
  4. padding-left
  5. padding

properti untuk border :

  1. border-top
  2. border-right
  3. border-bottom
  4. border-left
  5. border
