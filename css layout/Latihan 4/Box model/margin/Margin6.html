lanjutan pembelajaran keempat

melakukan shorthand margin yaitu dengan cara memakai properti margin.

penjelasan 4 value margin.

  1. jika ditulis 1 value,akan buat keempat sisinya sama
  2. jika ditulis 2 value :

    3. value 1 untuk atas dan bawah

    value 2 untuk kiri dan kanan

  3. jika ditulis 3 value :

    4. value 1 untuk atas

    value 2 untuk kiri dan kanan

    value 3 untuk bawah

  4. jika ditulis 4 value :

    5. value 1 untuk atas

    value 2 untuk kanan

    value 3 untuk bawah

    value 4 untuk kiri

ini adalah penggunaan shorthand margin

ini adalah contoh shorthand margin dengan 1 value.

1

ini adalah contoh shorthand margin dengan 2 value.

2

ini adalah contoh shorthand margin dengan 3 value.

3

ini adalah contoh shorthand margin dengan 4 value.

4

untuk penggunaannya bisa ditambahkan auto,tapi khusus yang kanan dan kirinya saja.

untuk contoh yang diambil ini adalah shorthand margin dengan 2 value yang ditambahkan auto dibagian margin kiri dan kanannya. :

5

kembali kecara pakai margin
