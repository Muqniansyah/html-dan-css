penjelasannya

overflow digunakan untuk mengatur perilaku elemen yang keluar dari parentnya

ada 4 value dari properti overflow,jenisnya yaitu :

  1. visible

    2. jika ada konten keluar dari parentnya,akan diperlihatkan kontennya.

  2. auto

    3. menambahkan scroll keyika elemen/konten tidak cukup.

  3. hidden

    4. jika konten keluar dari parentnya, akan disembunyikan kontennya.

  4. scroll

    5. memunculkan scroll ketika elemen/konten tidak cukup
