ini pembelajaran keenam tentang pseudo class

pseudo class yang berhubungan dengan link

  1. :link
  2. :hover
  3. :active
  4. :visited

pseudo class yang berhubungan dengan posisi elemen(1) atau pseudo class yang memiliki parent.

  1. :first-child
  2. :last-child
  3. :nth-child

pseudo class yang berhubungan dengan posisi elemen(2) atau pseudo class yang tidak memiliki parent.

  1. :first-of-type
  2. :last-of-type

contoh dari pseudo class tipe 1 yg berhubungan dengan link:

Hello World

contoh dari pseudo class tipe 2 yg memiliki parent:

  • link 1

  • link 4
  • link 5
  • link 6
  • link 7
  • link 8
  • link 9
  • link 10

untuk memilih elemen pertama dari sebuah jenis/tipe tag, yaitu :first-of-type

untuk memilih elemen terakhir dari sebuah jenis/tipe tag, yaitu :last-of-type
