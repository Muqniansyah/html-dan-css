ini pembelajaran kedua tentang font

Di font ini ada :

  1. font-family
  2. font-size
  3. font-weight
  4. font-variant
  5. font-style
  6. line-height

untuk font-size ada 3 jenis: px, %, em

untuk font-weight ada 5 jenis: lighter, normal, 100-900, bold, bolder

untuk font-variant ada 2 jenis: normal, small-caps

untuk font-style ada 3 jenis: normal, italic, oblique

untuk line-height ada 3 jenis: normal, px, em
