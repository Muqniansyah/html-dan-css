ini pembelajaran ketiga tentang text

Di text ini ada:

  1. color
  2. text-align
  3. text-indent
  4. text-decoration
  5. text-transform
  6. letter-spacing
  7. word-spacing

untuk warna kita bisa memberikan 3 tipe warna yaitu :

  1. nama warna
  2. hexadecimal
  3. rgb

untuk text-align ada 4 jenis: left, right, center, justify

untuk text-indent ada 2 jenis: px, %

untuk text-decoration ada 4 jenis: none, underline, overline, line-through

untuk text-transform ada 4 jenis: none, lowercase, uppercase, capitalize

untuk letter-spacing ada 2 jenis: normal, px

untuk word-spacing ada 2 jenis: normal, px
