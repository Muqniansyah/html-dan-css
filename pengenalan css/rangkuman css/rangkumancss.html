rangkuman pembelajaran 1

embed adalah tipe penyimpanan css yang langsung dalam html yang berada di head.

external adalah tipe penyimpanan css yang tidak langsung dalam html. dengan cara memiliki file sendiri dengan ektensi file css, dan nanti dihubungkan dengan cara link.

inline adalah tipe penyimpanan css yang langsung dalam html yang berada di body. tipe inline ini kurang disarankan karena memakai css seharusnya digunakan terpisah antara html dan css. tujuannya supaya mempermudah dan lebih simple.



rangkuman pembelajaran 2

ini pembelajaran kedua tentang font

Di font ini ada :

  1. font-family
  2. font-size
  3. font-weight
  4. font-variant
  5. font-style
  6. line-height

untuk font-size ada 3 jenis: px, %, em

untuk font-weight ada 5 jenis: lighter, normal, 100-900, bold, bolder

untuk font-variant ada 2 jenis: normal, small-caps

untuk font-style ada 3 jenis: normal, italic, oblique

untuk line-height ada 3 jenis: normal, px, em

  1. color

  3. text-indent
  4. text-decoration

untuk word-spacing ada 2 jenis: normal, px



rangkuman pembelajaran 4

ini pembelajaran keempat tentang background

Di background ini ada:

  1. background color
  2. background image
  3. background repeat
  4. background position

untuk background colornya bisa memakai 3 tipe warna, yaitu: warna, hexadecimal, rgb.

untuk background-repeat ada 4 jenis: repeat, repeat-x, repeat-y, no-repeat

posisi untuk background-position ada 2 yaitu:

  1. terhadap sumbu x

    2. yaitu : left/center/right

  2. terhadap sumbu y

    3. yaitu : top/center/bottom

dan itu masih bisa di kombinasi antara posisi untuk sumbu x dan y



rangkuman pembelajaran 5

ini pembelajaran kelima tentang selector dasar

selector digunakan pada css untuk mengenali sebuah elemen html yang akan diberi style.

yang bisa digunakan diselector yaitu: elemen html, id, class, complex selector. untuk yang id dan class itu adalah selector buatan sendiri, yaitu selector khusus.

untuk tipe elemen html

operator yang pertama adalah koma(,) yang digunakan untuk 2 elemen yang ingin diberikan style yang sama, caranya cukup menuliskan 1 properti/diberikan grouping dengan operator koma.koma bisa diartikan sebagai dan.

operator yang kedua adalah spasi yang digunakan untuk yang lebih spesifik. spasi artinya didalam.

perbedaan dari id adalah:

  1. sebuah elemen html hanya boleh punya 1 id.
  2. setiap halaman hanya boleh punya 1 elemen dengan id tersebut atau id tidak boleh sama.
  3. dapat digunakan penanda untuk link
  4. digunakan untuk javascript
  5. sebaiknya tidak digunakan untuk css(lebih baik gunakan class) karna bebannya lebih berat.

rangkuman pembelajaran 7

inheritance adalah sebuah elemen mewarisi beberapa nilai dari properti yang dimiliki oleh elemen parentnya.

properti yang diwariskan :

  1. color
  2. font
  3. letter-spacing
  4. line-height
  5. list-style
  6. text-align
  7. text-indent
  8. text-transform
  9. visibility
  10. white-space


rangkuman pembelajaran 8

ini pembelajaran kedelapan tentang specificity.

specificity yaitu setiap deklarasi (selector) memiliki berat yang berbeda. berat tersebut menentukkan seberapa spesifik sebuah elemen dapat dipilih oleh selector.
